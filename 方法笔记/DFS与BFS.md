### 回溯法
> 回溯法本身是基于DFS实现的，只是回溯节点时也要额外回溯状态
* 回溯法的标准模板:
    ```
    result = []
    def backtrack(路径, 选择列表):
        if 满足结束条件:
            result.add(路径)
            return

        for 选择 in 选择列表:
            做选择
            backtrack(路径, 选择列表)
            撤销选择```
* 回溯本质上也是DFS，最好按照DFS不重不漏的思路遍历状态
* 在回溯的过程中不能修改传入的path变量本身，最好在递归时使用"path+a"表达式
* 回溯时不光回溯节点，所有的状态都需要回退
* 在特殊情况下需要注意对返回结果的去重，需要设计恰当的剪枝原则
